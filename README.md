# Верстка с использованием библиотеки Bootstrap

Сайт со скачанными с https://tululu.org/ книгами. Количество книг и, соответсвенно, количество страниц на сайте можно менять.

На одной странице размещается не более 20 книг.

![Пример страницы](https://github.com/atskayasatana/Images/blob/main/index.png)

Чтобы открыть текстовую версию книги нужно кликнуть на "Читать":

![Читать](https://github.com/atskayasatana/Images/blob/main/book_example.png)


## Скачивание книг

Для скачивания книг используются функции [Отсюда](https://github.com/atskayasatana/Parsing_2), пользователю нужно определится с количеством страниц для скачивания и расположением файлов.
